# Анализ данных сервиса аренды самокатов (kicksharing data analysis project)

## Цель проекта

Провести полный цикл аналитики данных сервиса аренды самокатов: подготовка данных, исследовательский анализ, метрики, сегментация, статистические гипотезы и бизнес-инсайты.

Проект направлен на выявление точек роста для продукта и построение рекомендаций, влияющих на выручку.

## Технологический стек

- **Python:** pandas, numpy, matplotlib, seaborn, scipy, statsmodels
- **SQL:** PostgreSQL, ipython-sql
- **Инструменты:** Jupyter Notebook, Git

## Структура проекта

Проект состоит из 4 ноутбуков, отражающих последовательность аналитического цикла.

### 1. Очистка данных (data cleaning)
- **Файл:** `1_data_cleaning.ipynb`
- Обработано 396 тыс. записей.
- Удалены аномалии и выбросы — датасет уменьшен на 11.2% для чистоты эксперимента.
- Выполнен feature engineering — добавлено 7 новых признаков.

### 2. Разведочный анализ данных (EDA)
- **Файл:** `2_exploratory_analysis.ipynb`
- Исследованы распределения, корреляции, сезонность.
- **Инсайт:** активность пользователей достигает пика в пятницу и плавно снижается в выходные. Наблюдается активность во второй половине каждого дня (14-18 часов), а в будни ещё выделяется утренний пик (5-6 утра).
- **Инсайт:** сезонность ярко выражена с пиком в августе-сентябре и спадом в октябре.

### 3. SQL-аналитика и метрики
- **Файл:** `3_sql_queries.ipynb`
- Написано 30+ SQL-запросов.
- Рассчитаны ключевые метрики: `DAU`, `WAU`, `MAU`, `ARPU`, `Revenue`, `Retention Rate`.
- Проведена сегментация пользователей: `one-time`, `casual`, `regular`, `heavy`.
- Построен когортный анализ (retention падает со временем).

### 4. Статистический анализ и гипотезы
- **Файл:** `4_metrics_and_hypotheses.ipynb`
- Проверены гипотезы: U-критерий Манна-Уитни, Хи-квадрат, корреляционный анализ Пирсона/Спирмена.

## Ключевые выводы и инсайты

### 1. Heavy users — драйвер выручки
> **13.5% пользователей генерируют 58.3% выручки.**

Средняя частота поездок у heavy users в ~11 раз выше, чем у остальных. Потеря даже малой части этого сегмента критична для бизнеса.

### 2. Портрет пользователя
Активные пользователи — мужчины 18-40 лет из крупных городов.
Они совершают на **32%** больше поездок на пользователя, чем женщины. Однако, женская аудитория является точкой роста: их поездки более длительные и дорогие (выше средний чек).

### 3. Женщины как точка роста
Средний чек у женщин выше, чем у мужчин. Однако retention ниже: 54% против 61% у мужчин.
Учитывая их высокий средний чек, это критический источник упущенной выгоды. Увеличение retention женщин хотя бы до уровня мужчин может дать прирост выручки (по оценке **~1.5-2%**).

### 4. Платежеспособность в выходные
Средняя стоимость поездки в выходные статистически значимо выше.
Пользователи готовы платить больше за поездку. Есть потенциал для внедрения динамического ценообразования.

### 5. Влияние возраста на чек
Корреляция близка к нулю.
Возрастная сегментация для маркетинга менее эффективна, чем поведенческая.

## Рекомендации для бизнеса

1.  **Фокус на удержание женщин:** разработать маркетинговую кампанию и промо-механики, нацеленные на женскую аудиторию, чтобы поднять их retention с 54% до уровня мужской аудитории (61%).
2.  **Работа с heavy users:** внедрить программу лояльности для топ-13.5% пользователей, генерирующих основную выручку, чтобы минимизировать их отток.
3.  **Тарифные эксперименты:** рассмотреть введение специальных тарифов для выходных дней, так как готовность платить в эти дни выше.
